# Customer Targeting Optimization using Uplift Modeling

## Problem
Companies send marketing emails to all customers, leading to wasted cost.

## Solution
Used uplift modeling to identify customers who respond due to marketing.

## Approach
- T-Learner (Two-model approach)
- Random Forest
- Uplift Score = Treatment - Control

## Results
- Improved targeting efficiency
- Identified high-value customers

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Scikit-uplift
